\section*{Задача}
Петя и Вася решили разметить $n$ объектов на несколько классов. Насколько у них получились согласованные метки?

Петя $i$-й объект отнес к классу $p_i$ ($0 \le p_i < n$). Вася $i$-й объект отнес к классу $v_i$ ($0 \le v_i < n$). Вычислите значение функции похожести набора меток:

\[
Similarity(p, v) = \dfrac{1}{2}\dfrac{\sum_{i=1}^{n} \sum_{j=1}^{i-1} I[p_i=p_j] \cdot I[v_i=v_j]}{n \cdot (n-1)}
\]

где $I[exp]$ принимает значение $1$, если выражение $exp$ истинно, и $0$ иначе.

Другими словами, значение $Similarity(p, v)$ равно доле пар различных индексов $(i,j)$, для которых и Петя, и Вася одинаково определили принадлежность одному классу.

\section*{Формат ввода}
\begin{itemize}
    \item В первой строке записано одно целое число $n$ ($2 \le n \le 1\,000\,000$).
    \item Во второй строке записаны $n$ целых чисел $p_1, p_2, \dots, p_n$.
    \item В третьей строке записаны $n$ целых чисел $v_1, v_2, \dots, v_n$.
\end{itemize}

\section*{Формат вывода}
Выведите ответ в виде несократимой дроби $P/Q$ (без пробелов).

\section*{Ограничения}
\begin{itemize}
    \item Ограничение времени: 10 с
    \item Ограничение памяти: 1 ГБ
\end{itemize}

\section*{Пример 1}
\begin{itemize}
    \item \textbf{Ввод:}
    \begin{verbatim}
3
1 2 0
0 2 1
    \end{verbatim}
    \item \textbf{Вывод:}
    \begin{verbatim}
1/1
    \end{verbatim}
\end{itemize}
